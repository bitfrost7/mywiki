# 数据库访问层 — Data Access Layer

## 1. 模块职责

本模块是 recycle 服务的**数据库连接与查询入口层**，社区 C5 + C12 共 2 个主要源文件 + 3 个自动生成的查询文件，负责：

- **数据库连接管理**：MySQL 连接初始化、连接池配置（`MaxIdleConns=100`, `ConnMaxLifetime=1h`）、日志适配
- **GORM 查询入口**：通过 GORM Gen 生成的强类型查询接口（`db/query/gen.go`）
- **读写分离**：支持通过 `ReadDB()` / `WriteDB()` 切换数据库读写节点（`db/query/gen.go:46-52`）
- **事务管理**：所有写操作均在 `Transaction()` 内执行（`db/db.go:94, 120, 170, 202`）
- **监控插件**：集成 Prometheus 监控中间件（`gorm/plugin.NewMonitor`），采集 DB 连接数和请求延迟

## 3. 关键实现逻辑

### 3.1 连接初始化流程（`NewDatabase` @ `db/db.go:46-91`）

```
NewDatabase(zapLogger, cfg)
  │
  ├── 1. zapgorm2.New(zapLogger)                     — 创建 GORM 日志适配器 (L47)
  │      └── logger.Context = extract()               — 注入 RequestUUID/SessionID 到日志 (L48)
  │      └── logger.IgnoreRecordNotFoundError = true   — 忽略 RecordNotFound 错误 (L49)
  │
  ├── 2. 根据日志级别设置 GORM LogMode (L51-60)
  │      ├── Debug/Info  → LogMode(Info)     — 打印 SQL
  │      ├── Warn        → LogMode(Warn)
  │      └── Error/以上  → LogMode(Error)
  │
  ├── 3. gorm.Open(mysql.Open(DSN), &gorm.Config{Logger: loggerI})  — 打开 MySQL 连接 (L62-64)
  │
  ├── 4. gormDb.Use(plugin.NewMonitor(...))           — 集成监控中间件 (L70-78)
  │      ├── DBStatCollectInterval: 10s               — 每10秒采集DB连接数
  │      ├── DBStatCollector:         xpro.ExportDBStats
  │      ├── DBBeforeMetricCollector: xpro.ExportDBBeforeMetric  — 请求计数
  │      └── DBAfterMetricCollector:  xpro.ExportDBAfterMetric   — 延迟/结果
  │
  ├── 5. sqlDB.SetConnMaxLifetime(ConnMaxLifetime)    — 连接池配置 (L85-86)
  │      └── sqlDB.SetMaxIdleConns(MaxIdleConns)
  │
  └── 6. return &Database{ db: query.Use(gormDb) }   — 绑定 GORM Gen 查询入口 (L88-90)
```

### 3.4 事务保证

所有写操作均在 `d.db.Transaction()` 内执行（`db/db.go:94, 120, 170, 202`）：

```go
d.db.Transaction(func(tx *query.Query) error {
    // tx 是事务内的 Query 实例，使用 tx.TVpcEndpoint / tx.TService / tx.TConnectInfo
    // 操作均为同一数据库连接
    // 任一 error 返回 → 全量回滚
    return nil  // nil → 提交事务
})
```

确保：多表更新间的原子性、故障时全量回滚、使用 `tx.clone(tx)` 创建事务内独立 Query 实例。

### 3.5 读写分离

通过 `dbresolver.Read` / `dbresolver.Write` clause 切换（`db/query/gen.go:46-52`）：
```go
func (q *Query) ReadDB() *Query  { return q.ReplaceDB(q.db.Clauses(dbresolver.Read)) }
func (q *Query) WriteDB() *Query { return q.ReplaceDB(q.db.Clauses(dbresolver.Write)) }
```
