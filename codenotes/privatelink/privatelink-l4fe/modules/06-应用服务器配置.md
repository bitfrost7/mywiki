## 数据库初始化 (`mygorm/database.go`)

### New() (行 25-52)

```go
func New(l *zap.Logger, logLevel string, cfg *Config) (*db.Query, error)
```

核心步骤：
1. 根据日志级别配置 GORM logger（debug 级别显示 Info，其他级别显示 Warn）
2. 使用 `zapgorm2` 桥接库将 zap 日志接入 GORM
3. 打开 MySQL 连接（`gorm.Open(mysql.Open(cfg.DSN))`）
4. 设置连接池：`MaxIdleConns=100`，`ConnMaxLifetime=1h`
5. 通过 `db.Use(gormdb)` 创建并返回 `Query` 实例

### NewGen() (行 54-74)

```go
func NewGen(conf *Config) error
```

生成 ORM 代码的核心逻辑：
1. 连接到 MySQL
2. 创建 `gen.Generator`（配置输出路径、without context mode、with query interface）
3. 为四张表调用 `GenerateModel()`：
   - `t_service`
   - `t_service_snatips`
   - `t_service_whitelist`
   - `t_vpc_endpoint`
4. 执行 `g.Execute()` 生成代码
