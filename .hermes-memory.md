## mywiki pipeline 关键发现（2026-06-29）

- `kanban_create()` tool 参数：skills 是 string array（`skills=["interface-sk"]`），不是单值字符串
- CLI 版：`hermes kanban create --skill interface-sk`
- Worker 有 **terminal、read_file、write_file** 等 35 个工具（不是只有 kanban 工具）
- Worker PWD 是 `/Users/user/.hermes/kanban/workspaces/<task_id>`，路径是 scratch 临时目录
- `$HERMES_PROFILE` 自动注入，可用作权限判断
- Model 不 follow SKILL.md 的指令，需要把关键操作（discuss init、创建子卡）移到 shell 脚本里做
- 设计文档：`~/Documents/Code/work/mywiki/docs/pipeline-v5-design.md`

## discuss 二进制

- 位置：`~/Documents/Code/work/mywiki/bin/discuss`
- 加密：AES-256-GCM，key 编译时嵌入
- 权限：`$HERMES_PROFILE` 控制命令权限
- 数据：`raw/assets/privatelink/<service>/discuss.enc`
- 重新编译：`cd /tmp/discuss && go build -o ~/Documents/Code/work/mywiki/bin/discuss .`
